# Social Media Tags Registry (SMT)

A responsive web application for managing and browsing social media handles of organizations in the medical technology zone.

## 🌐 Live Demo
[https://apmedtechzone.github.io/smt/](https://apmedtechzone.github.io/smt/)

## 🚀 Features
- **Responsive Design** - Works on mobile and desktop
- **Local Storage** - Data persists in your browser
- **Multi-list Management** - Organize tags into custom lists
- **Search & Filter** - Find organizations quickly
- **Copy to Clipboard** - One-click tag copying
- **Suggest New Entries** - User contribution system
- **Admin Panel** - Manage approvals and data (login: admin/admin123)

## 🛠️ Technology Stack
- **HTML5** + **CSS3** (Bootstrap 5)
- **Vanilla JavaScript**
- **LocalStorage** for data persistence
- **Font Awesome** for icons

## 📱 Usage
1. Open the live site in any browser
2. Browse existing social media tags
3. Use search and filters to find specific organizations
4. Click copy icons to copy all visible tags in a column
5. Suggest new entries using the "Actions" menu
6. Admins can login to manage data and approvals
